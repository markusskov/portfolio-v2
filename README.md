## Getting Started

### Installing

To use this project:

1. Clone the repo:

```bash
git clone https://github.com/markusskov/portfolio-v2.git
```

2. Install the dependencies:

```
yarn install
```

### Running

To run the app, run the following commands:

```bash
yarn run dev
```
